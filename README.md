# SkyUnit Frontend

React + TypeScript + Vite frontend for SkyUnit Real Estate booking platform.

## Features
- User authentication with JWT
- Booking management interface
- Dashboard with booking history
- Real-time status updates
- Integration with Supabase backend via Render API

## Environment Variables
```
VITE_API_BASE_URL=https://your-render-api.onrender.com
```

## Installation
```
npm install
npm run dev
```

## Build
```
npm run build
```
